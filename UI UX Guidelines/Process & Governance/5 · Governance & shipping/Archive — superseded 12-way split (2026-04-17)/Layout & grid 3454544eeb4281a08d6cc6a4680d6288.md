## Spacing and sizing (NN/g and Material Design)

- **Grid:** Strict **8pt** grid for layout spacing (8, 16, 24, 32, 48, 64).
- **Micro-adjustments:** **4pt** grid for inner component spacing.
- **Touch targets:** Minimum **44×44px** for all interactive elements.

## Approved layout patterns

### List pages

- Header: title + primary CTA + filters/search
- Body: table for dense data; cards for summaries
- Details: drawer preferred for row click; separate page for deep workflows

### Detail pages

- Summary header: key metadata + primary/secondary actions
- Sections: grouped; collapsible if long

### Forms

- Inline validation preferred.
- Clear error copy: what happened + how to fix + how to recover.

## Context-specific hierarchy

Where a surface differs (e.g. **dense admin** vs. **marketing card**), document **type scale and density** in Figma for that pattern — not only in prose.
