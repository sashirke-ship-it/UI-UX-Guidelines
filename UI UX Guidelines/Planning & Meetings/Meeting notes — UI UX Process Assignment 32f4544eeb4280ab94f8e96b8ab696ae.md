### Assignment Context

Jorge assigned the team the task of establishing a UI/UX process for the organization because none currently exists.  The goal is to guide the team through creating a feasible UI/UX process that fits the current situation, resources, build workflows, and tech stack. 

### Research Focus Areas

- Latest tools and methods allowing non-designers and non-technical people to build prototypes and create sound UI/UX decisions
- Wireframing solutions that don't require spending excessive time on Figma
- Progress from "we don't have anything" to "we should have something" to evaluating best options to determining what fits the organization

### Agentic Design Approach

Jorge introduced the concept of agentic design, which works similarly to backend development for design.  This involves creating a design MD file — a design system described in a document that an agent can pick up and use in Cursor to ensure consistent UI/UX experience across all projects. 

### Current Gap

No projects currently have prototyping or wireframing, including the prediction markets product.  The team has an opportunity to take project documentation and create visual prototypes or wireframes using tools like Stitch or through Cursor. 

### Urgency & Next Steps

Jorge emphasized this work can't wait and needs to be completed quickly.  The team should get started immediately, work extensively with AI, and reach out if they get stuck or need clarification.
