## Assignment summary (from meeting)

- **Gap:** There is **no mature UI/UX process** today; most work (including **prediction markets**) lacks prototyping / wireframing.
- **Ask:** Progress from *nothing* → *we need something* → *evaluate options* → *pick what fits us*.
- **Materials:** Use the **latest materials Jorge shared** — run them through AI, implement understanding, produce **mockups** where docs already exist.
- **Agentic angle:** Treat **design like backend for agents** — a **`design.md` / design-system-in-docs** so agents (e.g. in Cursor) deliver **consistent UI** across projects.
