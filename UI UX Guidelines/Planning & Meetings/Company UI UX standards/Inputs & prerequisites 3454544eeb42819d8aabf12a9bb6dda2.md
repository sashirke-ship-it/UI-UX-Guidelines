# Inputs & prerequisites

## Before you customize this hub

Use this checklist to gather inputs once; link artifacts from your drive or wiki.

### Brand and marketing

- [ ]  Latest **brand guidelines** (PDF or URL)
- [ ]  Logo usage, color palette, and any restricted pairings
- [ ]  Photography / illustration style guide

### Product and UX

- [ ]  Inventory of **products** and **surfaces** (web, mobile, email, kiosk…)
- [ ]  Top **5–10 user journeys** per major product
- [ ]  Known **pain points** from support or research (optional)

### Engineering and tooling

- [ ]  Front-end stack (**React / Vue / native / other**)
- [ ]  Design tool (**Figma** library URL)
- [ ]  **Storybook** or component catalog link (if any)
- [ ]  **CSS variables / theme** approach for tokens

### Compliance and risk

- [ ]  Industry regulations (**HIPAA, PCI, GDPR, COPPA**, etc.) if applicable
- [ ]  **Security** constraints (PII display, masking, retention)
- [ ]  **Localization** requirements (RTL, number/date formats)
